# Deployment Notes

Use Vercel for this app.

This is a Next.js modular application. GitHub Pages is only suitable for static HTML demos and should not be used for the live app.

## Vercel Toolbar

If the floating Vercel toolbar appears on the live site, disable it in Vercel:

Project → Settings → General → Vercel Toolbar → Off

If the UI location changes, search Vercel settings for "Toolbar".
