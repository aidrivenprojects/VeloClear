# VeloClear Roadmap

## Current State

The current product is a visual and structural MVP. It has:
- discovery entry
- dashboard
- projects
- lifecycle pages
- connected delivery graph explanation
- Supabase persistence foundation

It does not yet fully implement:
- real document ingestion
- AI extraction backend
- persisted delivery graph objects
- live integration with Jira / ADO / MSP
- predictive intelligence

## Next Stable Build

Build only the connected delivery graph MVP.

Scope:
- project lifecycle navigation
- phase contextual tools
- product backlog
- sprint backlog
- sprint execution
- blocker tracking
- risk trigger
- issue creation
- escalation
- decision
- resolution
- retrospective learning

## Success Criteria

The app must demonstrate one full trace:

```text
PBI blocked
→ Sprint slips
→ Risk triggered
→ Issue opened
→ Escalation created
→ Decision logged
→ Resolution captured
→ Retrospective action stored
→ Future recurrence checked
```

If this works, VeloClear becomes a real Delivery Intelligence OS MVP.
